# JC-OM

Independent builder of local-first AI systems, developer tools, and efficient
LLM inference.

I turn ambitious prototypes into software that can be inspected, tested, and
used on real machines. My work is open-source by default and focuses on clear
trust boundaries, reproducible evidence, and practical human control.

## Featured work

I am preparing the first public releases under
**[JCOM Labs](https://github.com/jcomlabs)**:

- **Code Hangar** — a local-first context engine for reusable project knowledge;
- **Axiomux** — a multi-provider agent harness with memory, skills, and fusion;
- **MoE Autopilot** — adaptive MoE expert placement for consumer GPUs;
- **Window Swap** — a focused Windows utility for snapped-window workflows.

## Engineering principles

- local-first when privacy or control matters;
- evidence before performance claims;
- reversible automation and explicit trust boundaries;
- small reproducible demos alongside ambitious systems;
- honest disclosure of limitations and AI-assisted development.

Based in Portugal. Building in public as **JC-OM**.

# JC-OM

Independent builder of local-first AI systems, developer tools, and efficient
LLM inference.

I turn ambitious prototypes into software that can be inspected, tested, and
used on real machines. My work is open-source by default and focuses on clear
trust boundaries, reproducible evidence, and practical human control.

## Featured work

My public projects live under **[JCOM Labs](https://github.com/jcomlabs)**:

- **[Axiomux](https://github.com/jcomlabs/axiomux)** — a multi-provider agent
  harness with governed memory, skills, effect-aware fallback, and fusion;
- **[Window Swap](https://github.com/jcomlabs/window-swap)** — a focused
  Windows utility for snapped-window workflows.
- **[Code Hangar](https://github.com/jcomlabs/code-hangar)** — a local-first
  context engine for inspecting AI-assisted projects across tools, sessions,
  files, and recorded changes.
- **[MoE Autopilot Studio](https://github.com/jcomlabs/moe-autopilot-studio)** —
  a Windows-first evidence lab for measured MoE placement decisions and the
  reproducible companion to my broader MoE Autopilot research.

The core MoE Autopilot research repository is being prepared for public release.

## Engineering principles

- local-first when privacy or control matters;
- evidence before performance claims;
- reversible automation and explicit trust boundaries;
- small reproducible demos alongside ambitious systems;
- honest disclosure of limitations and AI-assisted development.

Based in Portugal. Building in public as **JC-OM**.
